## Git 명령어 내용 정리

| 명령어    | 기능                                                                                 |
|----------|--------------------------------------------------------------------------------------|
| `git init`   | 저장소 초기화 - 프로젝트 시작 직 딱 한 번 입력 <br>(정확한 프로젝트 경로에서 입력해야 함) |
| `git status` | 저장된 상태 확인 명령어                                                          |
| `git add`    | 변경한 파일을 지정                                                               |
| `git commit` | 파일을 실제로 저장                                                               |
| `git log`    | 코드 저장 이력 확인                                                              |
| `git push`   | GitHub에 반영                                                                    |
| `git clone`  | GitHub에 있는 코드 복사                                                          |
| `git pull`   | 다른 사람이 변경한 코드를 내 코드에 가져오기                                     |
